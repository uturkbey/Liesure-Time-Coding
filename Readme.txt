This program is for calculating the maximum sum of the numbers along a path in a triangular form according to given rules below;
 1. You will start from the top and move downwards to an adjacent number as in below.
 2. You are only allowed to walk downwards and diagonally.
 3. You can only walk over NON PRIME NUMBERS.
 4. You have to reach at the end of the triangle as much as possible.
 5. You have to treat your input as triangle.

To use the program:
1)Store a txt data file including a triangular pyramide shape with no zeros and only integer elements.
2)Compile and run the java file.
3)Type the name of the input file with ".txt" part at the end.
4)Maximum sum based on the regulations will be printed on the console.
